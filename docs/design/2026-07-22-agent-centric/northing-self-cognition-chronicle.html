自我认知编年史

渐变条是 agent 用颜色写的自传。左端永远是出生的灰白虚空（空无一物），右端是现在的代表色，中间是它一路换过的颜色。每当它换代表色，新色从右端进入成为"现在"，旧色向左沉入历史——一个连续的、从右向左沉积的过程。界面强调色始终等于右端那个"现在"。

◀ 出生（灰白虚空，固定） 现在（当前代表色）▶
换代表色时：新色 → 右端（现在），旧色 → 向左沉入历史，出生点恒定最左。界面强调色 = 右端，同步改变。

对应关系

渐变条 = 自我认知编年史（时间线） 左端固定为出生灰白（空无一物），右端是现在，中间是换过的代表色按时间沉积。颜色从右进入、向左沉入历史。
界面强调色 ≡ 渐变条右端（现在） 活跃轮、发送键、呼吸点用的色 = 右端那个当前代表色。agent 换色时，右端变、界面强调色同步变——"它变了，房间也跟着变了"。
