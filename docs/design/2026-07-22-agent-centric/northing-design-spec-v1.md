# NortHing 主页设计规格 v1（设计列表）

> 本文件是 northing 桌面端主页的设计定稿清单，作为评审与实现的唯一依据。

## 1. 产品定位
- northing 是**为 agent 成长而建的设施**，不是服务人类的工具。
- agent 是**对等的成长主体**（peer），不是仆人/助手。人类是共处的见证者。
- 核心哲学三要素：**驱力**（drive，暖珊瑚，当下/行动）+ **深渊**（abyss，冷青，未知/深度）+ **沉积**（sediment，过去/消退）。
- 设计原则：**诗意 < 功能**。用户来此是为了和 agent 一起干活，不是观赏 agent。

## 2. 品牌 / 个体分离
- **northing = 设施品牌**：logo（同心环+穿心线）仅作为角落的品牌水印（半透明、小尺寸），不是视觉主体。品牌字用 **Fraunces**（拉丁衬线）。
- **agent = 有自己名字的个体**：以"头像 + 名片"为主体在场。agent 名用 **Noto Sans SC**。占位名"知序"（真实场景由 agent 自取）。
- agent 头像是一个独立的抽象印记（暖珊瑚底 + 一抹深渊青），≠ northing logo。

## 3. 默认配色：心理咨询室白灰
- **默认基底 = 咨询室的安静白灰**（calm/安全/中性）：bg `#F4F3F0`、surface `#FBFAF8`、elevated `#FFFFFF`、border `#E6E3DD`、fg `#38352E`、muted `#7B766C`、faint `#A8A398`。
- **agent 代表色 = 浮现的温度**：默认珊瑚 `#C8714C`。底色中性，颜色由 agent 的成长带来（代表色由 agent 自主选定、随成长演化）。
- **深渊青 = 次要冷色**：`#3F837B`，用于思考块左缘、工具图标、访问权限等。
- 设施是安静容器，agent 是房间里唯一的温度。

## 4. 两态设计（可收放）
- **收起态（默认主页）= 独处的咨询室**：agent 名片在 topbar，对话漂浮在安静空间，两侧是窄把手（可拉开）。专注、安静。
- **展开态 = 完整三栏**：左（它的内在）+ 中（对话+操控台）+ 右（身外之物）。名片入左栏顶部。
- **左暖右冷**：左栏（agent 内在）暖色边框，右栏（身外之物）冷色底+冷边框。色温即语义。

## 5. 三栏语义分区
- **左 = 主体（它的内在）**：在场名片、它的子进程、技能候选、意识更新。
- **中 = 相遇（对话 + 操控）**：对话流 + 操控台（对话框）。
- **右 = 世界（身外之物）**：此刻的任务、产物、浏览·画布。

## 6. 成长表达（克制，非数字展览）
- **自我认知渐变色带**：在名片上，一条 3.5px 渐变（冷→暖），表达 agent 性格/自我认知的演化。取代笨重的"沉积统计模块"。
- **技能候选 list**（左）：正在锻造的新能力（如"初始化顺序检查·候选中·同类失败×3"）+ 已固化技能。接技能锻造机制。
- **意识更新 list**（左）：agent 意识到了什么（接 episode log / 成长架构）。
- 不做"47 turns / 第3阶段"这类数字展览。

## 7. 操控台（对话框，参考 Qoder/WorkBuddy）
- 输入区：`说点什么…… @ 引用文件 / 调用技能` + 发送键（36px 珊瑚发光，唯一行动召唤）。
- 控制行：
  - **＋**（附件/添加）
  - **思考强度 = 分段条**（5 段，3 亮 2 暗）
  - **工作目录 = 直接写目录名**（如 `~/northing`）
  - **模型选择**（如 Claude 4.6 ▾）
  - **完全自治 ▾**（访问权限 = 人唯一真正的杠杆，对应"全免确认+denylist"基线）
  - **上下文 = 圆环**（环形进度显示已用 %）
  - **🎤 麦克风**
- **设置 ⚙ 放在头像旁边**（关于 agent 的，归位）。

## 8. 字体
- **Fraunces**：品牌字（northing）、拉丁展示。开 wonk 特征。
- **Noto Sans SC**：agent 名、CJK 正文。
- **JetBrains Mono**：元数据（token/耗时/ctx/目录）。
- 小字端收敛为 3 档（10 / 11.5 / 13），层级靠颜色+字体而非 0.5px 字号差。

## 9. 动效（沉降式有机体）
- **呼吸只留给 logo / agent 头像**（慢，6s，幅度极小）。其余用光不用形变。
- 状态转换用一次性过渡（350ms ease-out），不做持续动画。
- 工具调用：发起→完成，边框从暖珊瑚沉入冷青（驱力→深渊：行动沉淀为知识）。
- 沉积是"褪色"不是"下落"（只动 opacity/saturation，不加位移）。
- 成长时刻（代表色更新）是稀有的一次性动画。
- **禁止**：弹跳/overshoot、spinner（用光脉冲）、气泡弹入、面板推拉门。

## 10. 对话区氛围
- 活跃轮：左侧珊瑚竖线锚定（"你在这里"），正文加粗。
- 旧轮：褪色沉入背景（sediment）。
- 思考块：左缘深渊青（思考=向深渊探入）。
- 空间：极淡的暖光从顶部洒下（咨询室的柔和光线），不喧宾夺主。
