## 代码风格观察

### 正面模式

```rust

pub(crate) mod analyze;  // 业务逻辑
pub(crate) mod format;   // 格式化
pub(crate) mod prompt;   // prompt 构造
pub(crate) mod tests;    // 测试

pub use analyze::*;
pub use format::*;
// ...
```

### 负面模式 (出现 3+ 次)

```rust
// pattern: tests.rs 双层 mod tests 包装
#[cfg(test)]
mod tests {       // <-- 内部 wrapper
    use super::*;
    // ...
}                // <-- 关闭 wrapper
```

加上 mod.rs 又有 `mod tests;` → E0428

### 公共 API 保持稳健

R5x 全部保留 `pub use super::*;` 模式，外部 import 路径不破坏。这点 step-3.7 做得很一致。
