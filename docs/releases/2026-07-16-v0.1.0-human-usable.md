# v0.1.0-human-usable — 首个可安装发布版

> 发布日期: 2026-07-16
> Commit: `c1b0c27`
> 适用平台: Windows / macOS / Linux (CLI 核心)

## 简介

这是 northing 首个面向普通用户的发布版。安装后即可使用 CLI（命令行界面）进行 AI 辅助编程、会话管理、工具调用等操作。

## 主要变化

### 代码质量
- **Dead code 清理**: 编译器警告从 1390 → 151 (减少 89%)
- **God-file 拆分**: 超过 750 行的 3 个巨大文件 (model_config_form/chat/render/question) 已拆分为模块化子文件
- **QClaw review 通过**: 9.3/10 APPROVED (13 维度 rubric 全 PASS)

### 依赖管理
- **Syntect 可选化**: 语法高亮依赖 `syntect` + `syntect-tui` 改为可选 feature，默认仍然开启但不再强制绑定 C 依赖
- **Feature gate 完整**: `#[cfg(feature = "syntax-highlight")]` 正确覆盖所有语法高亮调用点
- **AGENTS.md 恢复**: 从 Mavis 迁移时期的 session 笔记恢复为原版英文技术文档 (203 行)

### 测试
- **932/933 测试通过** (per-crate + workspace 排除 Slint 桌面)
- 1 个已知 pre-existing 失败 (`turn_batch` B-3 问题，非本次引入)

## 安装

```bash
# 从源码构建
git checkout v0.1.0-human-usable
cargo build -p northhing-cli --release
```

## 已知限制

- **桌面 GUI** (Slint + Tauri): 尚未在所有平台上验证
- **Windows MSYS2 GCC**: workspace-level 测试需排除 `northhing` crate (feature unification 导致的二进制兼容性问题)
- **Mobile Web**: 尚未完全验证

## Roadmap (post-v0.1.0)

- Slint 桌面 C5 运行时修复
- 跨平台 installer 验证 (NSIS/DMG/deb)
- CI/CD pipeline
- MCP server 集成验证

# v0.1.0-human-usable — First Installable Release

> Release date: 2026-07-16
> Commit: `c1b0c27`
> Platforms: Windows / macOS / Linux (CLI core)

## Summary

This is the first end-user–focused release of northing. After installation, users can start the CLI for AI-assisted coding, session management, tool invocation, and more.

## Major Changes

### Code Quality
- **Dead code cleanup**: Compiler warnings 1390 → 151 (89% reduction)
- **God-file splits**: 3 oversized files (model_config_form/chat/render/question) split into modular sub-modules
- **QClaw review approved**: 9.3/10 APPROVED on 13-dimension rubric

### Dependency Management
- **Syntect optionalization**: `syntect` + `syntect-tui` moved to optional `syntax-highlight` feature; default build still includes it but no longer hard-binds the C dependency
- **Feature gating**: All syntax-highlight call sites properly gated with `#[cfg(feature = "syntax-highlight")]`
- **AGENTS.md restored**: Recovered from Mavis migration artifact back to original English technical document (203 lines)

### Testing
- **932/933 tests pass** (per-crate + workspace excluding Slint desktop)
- 1 pre-existing known failure (`turn_batch` B-3 issue, not introduced this release)

## Installation

```bash
# Build from source
git checkout v0.1.0-human-usable
cargo build -p northhing-cli --release
```

## Known Limitations

- **Desktop GUI** (Slint + Tauri): Not yet verified on all platforms
- **Windows MSYS2 GCC**: Workspace-level tests require `--exclude northhing` (feature unification binary compatibility issue)
- **Mobile Web**: Not yet fully verified

## Post-v0.1.0 Roadmap

- Slint desktop C5 runtime fix
- Cross-platform installer verification (NSIS/DMG/deb)
- CI/CD pipeline
- MCP server integration verification
